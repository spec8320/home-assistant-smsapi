# Przykładowe automatyzacje

Przed użyciem zmień identyfikatory encji oraz numery telefonów na własne.

- `alarm_garage.yaml` — ruch w garażu przy uzbrojonym alarmie,
- `garage_door_open.yaml` — brama otwarta przez 10 minut,
- `washing_machine_finished.yaml` — zakończenie pracy pralki,
- `ups_on_battery.yaml` — przejście UPS-a na baterię,
- `internet_down.yaml` — brak połączenia z internetem.
